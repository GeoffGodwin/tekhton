Result: 0+0+20+20+0+15 = 55/100 for Code Quality on a project with no code. Fix: change all three initializations to `0` and set them to their default-pass values (`20`/`20`/`15`) at the *top* of the `[[ -n "$sample_files" ]]` block, so they're only awarded when there's actual code to assess.
