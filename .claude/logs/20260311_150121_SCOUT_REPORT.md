## Key Design Decisions

1. **Model change only.** Milestone 1 acceptance criteria are model default (`opus`) + template depth. This is NOT multi-phase interview (Milestone 2), NOT depth-scoring completeness (Milestone 4), NOT generation prompt overhaul (Milestone 3).

2. **Template output format unchanged.** The parser `_extract_template_sections()` extracts `NAME|REQUIRED|GUIDANCE`. Milestones 2 and 4 will extend this (add PHASE marker, depth scoring). Milestone 1 templates just add more sections with better guidance — no parser changes needed.

3. **Test updates are part of M1.** Acceptance criteria say "All tests pass." The config tests hardcode `sonnet` and will fail once defaults change. Updating them is in scope.

4. **No prompt changes yet.** The interview prompt (plan_interview.prompt.md) and generation prompt (plan_generate.prompt.md) are not touched in M1. Better templates alone will improve interview depth. Prompt overhaul is Milestone 3.
