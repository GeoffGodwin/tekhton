- `report_retry` uses an em-dash (`—`) in the format string (`"Attempt ${attempt}/${max} — retrying in ${delay}s..."`). On a non-UTF-8 terminal the box chars fall back to ASCII, but the em-dash in the message string does not. Low impact since the message is informational, but for strict ASCII safety the em-dash could be replaced with a hyphen in the non-UTF-8 branch.
