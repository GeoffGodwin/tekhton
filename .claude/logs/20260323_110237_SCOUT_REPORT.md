## Suspected Root Cause Areas

- **Path detection vs. file exclusion:** `.cursor/` can contain large binary state files. Must filter to .md/.json/.yaml only to avoid bloat and false detection noise.
- **CLAUDE.md provenance ambiguity:** A project may have a hand-written CLAUDE.md (valuable to merge) OR a Tekhton-generated one (skip). Must check for `<!-- tekhton-managed -->` markers to distinguish.
- **Reinit config preservation:** Reinit path must NOT destroy existing pipeline.conf customizations. Must read → merge detections → write with VERIFY markers on changed values.
- **Git interop:** Tidy operation with optional commit requires robust error handling. Must handle repos without git, repos with staged changes, etc.
- **Interactive fallback:** In CI/headless mode, ARTIFACT_HANDLING_DEFAULT must be respected. Must support non-interactive mode via environment variable.
- **Merge agent invocation:** The merge agent is a new invocation pattern within init (not within pipeline execution). Must handle model selection, turn budgeting, output parsing from agent logs.
- **Synthesis context injection:** MERGE_CONTEXT.md must be read + injected into _assemble_synthesis_context(). Must handle when file doesn't exist gracefully.
