## Suspected Root Cause Areas

- **Prompts not wired up for repo map**: coder.prompt.md, reviewer.prompt.md, tester.prompt.md, scout.prompt.md, architect.prompt.md have no {{IF:REPO_MAP_CONTENT}} blocks yet
- **Stages not calling run_repo_map()**: coder.sh, review.sh, tester.sh, architect.sh still only inject ARCHITECTURE_CONTENT
- **Context budget doesn't account for repo map**: lib/context.sh `log_context_report()` doesn't measure or log repo map size
- **Missing test file heuristics**: tester.sh needs logic to map foo.py → test_foo.py, foo.ts → foo.test.ts
- **No reverse dependency lookup**: review.sh needs to find callers of changed files (PageRank top-N)
- **Template variables not exported**: lib/prompts.sh doesn't export REPO_MAP_CONTENT, REPO_MAP_SLICE
- **Graceful degradation incomplete**: stages should log warnings when REPO_MAP_ENABLED but indexer unavailable, then fall back to ARCHITECTURE_CONTENT
