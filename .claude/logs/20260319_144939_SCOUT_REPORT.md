## Suspected Root Cause Areas
1. **lib/hooks.sh:121** — `echo "$diff_stat" | head -n -1` only works on GNU systems. macOS head doesn't support `-n -1` syntax (which means "all but last line"). Replacement: `echo "$diff_stat" | awk 'NR>1{print prev} {prev=$0}'` which is portable to BSD/macOS.
2. **lib/common.sh:184** — `grep -oE '[0-9]+(\.[0-9]+)?%' | head -1 | tr -d '%'` silently returns empty string if claude usage output format changes (e.g., from "Session usage: 95%" to a different format). Current code treats empty extract as 0%, which is unsafe. Needs comment documenting expected format: "expects a line containing 'N%' or 'N.N%' from `claude usage` output".
