## Simplification

### S1 — Extract `_process_fifo_line()` helper in `lib/agent_monitor.sh` (lines 132–222)

**File:** `lib/agent_monitor.sh`
**Lines in scope:** 132–222 (the `while true` FIFO reader loop inside the subshell)

**Problem:**
The timed-read branch (lines 134–165) and the blocking-read branch (lines 211–218)
both process each line from the FIFO, but they have diverged:

| Feature | Timed branch | Blocking branch |
|---|---|---|
| Echo to tee (`>&3`) | Yes | Yes |
| Track `_last_line` | Yes | Yes |
| Ring buffer update | **Yes** | **No** |
| API error detection | **Yes** | **No** |
| python3 text print | Yes | Yes |

Future features added to the timed branch (e.g., turn-count extraction, structured
event detection) will require manual mirroring to the blocking branch every time.

**Remediation:**
Extract a plain bash function `_process_fifo_line()` defined immediately before the
`while true` loop (still inside the subshell `(…)` block). The function must:

1. Receive the line as `$1`
2. Append it to the ring buffer: `_rb[$(( _rb_idx % _rb_size ))]="$1"; _rb_idx=$(( _rb_idx + 1 ))`
3. Run the API error detection `case` block (verbatim from lines 142–160, replacing
   `"$line"` with `"$1"`)
4. Run the python3 text-print block (verbatim from lines 161–165, replacing
   `"$line"` with `"$1"`)

The function can access `_rb`, `_rb_idx`, `_rb_size`, `_stream_api_error`, and
`_stream_api_type` as subshell-globals — bash functions share the variable scope of
their enclosing subshell (no `local` needed, no array-passing problem).

After extracting the function, replace the per-branch inline logic:

**Timed branch** — remove lines 138–165, replace with:
```bash
_process_fifo_line "$line"
```

**Blocking branch** — remove lines 214–218, replace with:
```bash
_process_fifo_line "$line"
```

The `echo "$line" >&3` and `_last_line="$line"` assignments stay inline in each
branch (they are branch-specific: the timed branch also updates `_last_activity`,
so the ordering matters). Only the shared processing moves into the helper.

**Scope constraint:** Touch only `lib/agent_monitor.sh`. No other files. No
behavioral changes — identical logic, deduplicated.

**Verification:** `bash -n lib/agent_monitor.sh` and `shellcheck lib/agent_monitor.sh`
must pass. All existing tests must pass.
