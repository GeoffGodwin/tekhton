## Implementation Details
The milestone archival function now performs three sequential passes:
1. **Pass 1 (AWK rewrite)**: Removes the entire [DONE] milestone block from CLAUDE.md (heading + body) by outputting `next` instead of inserting a summary line
2. **Pass 2 (Pointer insertion)**: Inserts `<!-- See MILESTONE_ARCHIVE.md for completed milestones -->` comment under the ### Milestone Plan heading if not already present
3. **Pass 3 (Blank line collapse)**: Collapses 3+ consecutive blank lines down to 2 to clean up formatting after removal

Helper functions handle edge cases:
- `_insert_archive_pointer()` prevents duplicate comments across re-runs
- `_collapse_blank_lines()` uses AWK to count consecutive blank lines and preserve up to 2
- Both functions use tmpfile + mv pattern for portability (no sed -i)
