## Architecture Notes
- **Archive Format**: MILESTONE_ARCHIVE.md is append-only with timestamps and section headers (e.g., `## Archived: 2026-03-17 — Adaptive Pipeline 2.0`)
- **CLAUDE.md Mutation**: After archival, completed milestone in CLAUDE.md becomes a single-line summary: `#### [DONE] Milestone N: Title` (full definition removed)
- **Milestone Extraction**: Use regex matching `^[[:space:]]*####[[:space:]].*\[DONE\].*Milestone` to find [DONE] milestones, then extract to next sibling heading (`####`) as boundary
- **Idempotency**: Archive functions must detect already-archived milestones (single-line summaries) and skip silently
- **Integration Timing**: Startup archival runs after CLAUDE.md is read but before main pipeline loop; post-commit archival runs after git commit succeeds and tag is created
