- **lib/plan.sh line 117** — Comment "Future milestones will add: Step 5" marks where milestone review should be integrated into `run_plan()` after `run_plan_generate()`.
- **stages/plan_generate.sh lines 80-91** — CLAUDE.md is already written to disk by the agent before review UI. Review UI will read and display it, but needs to support [e]dit → re-read after editor closes, and [r]e-gen → overwrite on disk.
- **No milestone extraction function** — Need to implement `_extract_milestones_from_claude()` to parse "## Milestone N: Title" headings from CLAUDE.md for display in numbered format.

- **No editor integration for CLAUDE.md** — Pattern exists in tekhton.sh (line 735) but needs wrapper function for plan phase. Must reload CLAUDE.md after editor closes for [e]dit option.
- **No next-steps printing** — After [y]es approval, need to print instructions to run `tekhton --init` next.
