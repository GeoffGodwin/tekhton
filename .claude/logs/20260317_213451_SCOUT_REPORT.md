- **Markdown parsing edge cases** — handling varied milestone heading formats (####, #####, with/without [DONE] markers, decimal numbers like 0.5)
- **Integration of archival calls** in tekhton.sh — ensuring archival runs after commit and before state clear, handling state file updates
